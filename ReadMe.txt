###  Snake Game

A classic Snake game implemented using Java and the Swing framework. Move the snake using arrow keys, eat food, and grow your snake while avoiding collisions.

**Features:**
- Basic gameplay mechanics
- Real-time user input handling
- Game over conditions

**Technologies Used:**
- Java
- Java Swing
- Keyboard Event Listeners

###  How To Run ###

   git clone https://github.com/your-username/java-projects.git
   cd java-projects
